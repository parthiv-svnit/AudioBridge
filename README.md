# AudioBridge

AudioBridge allows you to stream audio from your Android device to your PC over the same network.

---

## 📦 Installation

1. Install the AudioBridge app on your Android device.
2. Open the app and provide the necessary permissions.

---

## 💻 PC Setup

1. On your PC, run:

   ```
   receiver.py
   ```

2. The IP address of your PC will be displayed in the terminal.

---

## 📱 Connect the App

1. Open the AudioBridge app.
2. Enter the IP address shown on your PC.
3. Click **Start Streaming**.

---

## 🌐 Important

- Make sure your **phone and PC are connected to the same network (WiFi)**.
- Phone's Hotspot (5Ghz) Recommended

---

## 🚀 Usage Flow

Install App → Run `receiver.py` → Enter PC IP in App → Start Streaming

---

## 🛠 Requirements

- Python installed on PC
- numpy and sounddevice package should be installed

   ```
   pip install numpy
   pip install sounddevice
   ```
- Same WiFi network connection
